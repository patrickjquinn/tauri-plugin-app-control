## Default Permission

Allows controlling app lifecycle (minimize, close, exit) and checking foreground state.

#### This default permission set includes the following:

- `allow-minimize-app`
- `allow-close-app`
- `allow-exit-app`
- `allow-is-app-in-foreground`
